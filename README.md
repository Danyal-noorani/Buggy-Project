# Buggy Project 2E10 Group W4
To get started install git from https://git-scm.com/install/

Follow installtion insturctions to login into your account and setup an empty folder, then
```
git init 

git remote add origin https://github.com/Danyal-noorani/Buggy-Project.git

git pull origin main
```
This will get you the latest version of the codebase

# To update codebase
Navigate to the directory where you initially ran  
```
git init 
```
Then
```
git add .
git commit -m "REASON/UPDATES for this commit"
git push -u origin main
```
